项目简介

本项目使用 icu_first24hours.csv（ICU 患者入院前 24 小时相关特征）进行数据清洗、描述统计与可视化， 并建立基线预测模型预测 HOSPITAL_EXPIRE_FLAG（院内死亡标记）。

数据来源

  • 数据集：icu_first24hours.csv（课程提供示例数据）
  • 样本量与字段说明：见报告 PDF

我做了什么（按评分点）

  1. 数据读取：使用 pandas 读取 CSV，检查数据维度、数据类型与缺失情况。
  2. 数据预处理：处理缺失值/特征筛选，划分训练测试集（分层抽样）。
  3. 统计分析：描述统计 + 可视化（目标分布、缺失值概览、分组箱线图）。
  4. 预测模型：建立基线模型（Balanced Logistic Regression；对照模型 Random Forest）。
  5. 模型评估与可视化：ACC、ROC-AUC、PR-AUC（AP），并绘制 ROC/PR 曲线。
  6. 项目展示网页：通过 GitHub Pages 公开展示项目与结果。
